# 工单查询结果排版模板

## 表格列定义

查询结果按以下表格格式展示：

| 工单ID | 创建时间 | 更新时间 | 用户昵称 | 处理状态 | 工单类型 | 产品 | 问题标签 | 问题内容 |
|--------|----------|----------|----------|----------|----------|------|----------|----------|

### 字段映射

| 显示列名 | API 字段 | 转换规则 |
|----------|----------|----------|
| 工单ID | `id` | 直接展示 |
| 创建时间 | `createTime` | 按"时间戳转换规则"格式化 |
| 更新时间 | `updateTime` | 按"时间戳转换规则"格式化 |
| 用户昵称 | `nick` | 直接展示 |
| 处理状态 | `processStatusInt` | 按"枚举值转换规则（处理状态）"转换 |
| 工单类型 | `typeInt` | 按"枚举值转换规则（工单类型）"转换 |
| 产品 | `product.name` / `productIdInt` | 优先展示 `product.name`；缺少名称时按产品枚举映射，未收录时显示原始产品 ID |
| 问题标签 | `questionTagIdInt` | 直接展示数值 |
| 问题内容 | `questionContent` | 截取前 50 字符展示，超出部分用 `...` 省略 |

---

## 时间戳转换规则

API 返回的时间字段可能有两种格式：

1. **Unix 时间戳（秒）**：使用 Python 转换
   ```bash
   python -c "import datetime; print(datetime.datetime.fromtimestamp(<TS>).strftime('%Y-%m-%d %H:%M:%S'))"
   ```

2. **日期字符串（已经是 `YYYY-MM-DD HH:mm:ss` 格式）**：直接展示

## 枚举值转换规则

### 处理状态

| 值 | 显示文本 |
|:--:|----------|
| 0 | 已完成 |
| 1 | 待处理 |
| 2 | 已废除 |
| 3 | 转研发 |
| 4 | 待处理加急 |
| 5 | 转研发已完成 |
| 6 | 产品待处理 |
| 7 | 转组长 |
| 8 | 转组长已完成 |

### 工单类型

| 值 | 显示文本 |
|:--:|----------|
| 1 | 普通咨询 |
| 2 | 投诉 |
| 3 | 退款 |
| 4 | 中差评 |
| 5 | 建议 |
| 6 | 发票 |
| 7 | 返现 |
| 8 | 操作咨询 |
| 9 | 产品跟进 |
| 10 | 故障 |
| 11 | 跨平台 |

### 产品

产品名称映射以 [工单接口文档的 `productIdInt` 枚举](workorder.md#productidint--产品) 为准。若响应包含 `product.name`，优先使用该字段；没有名称且 ID 未收录时，显示原始产品 ID。

## 分页摘要格式

每次查询结束后输出：

```
当前第 X 页 / 共 Y 页，共 Z 条记录
```

其中：`Y = ceil(total / pageSize)`

### 使用示例

```
当前第 1 页 / 共 15 页，共 150 条记录
如需查看更多，请告诉我页码。
```

---

## 空结果提示

当查询无结果时，输出：

```
查询时间范围内无工单记录，请调整查询条件。
```
