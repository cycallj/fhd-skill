# 弹幕捕手企微客户查询结果模板

## 表格列定义

默认按以下表格展示客户列表：

| 客户 ID | 创建时间 | 客户名称 | 微信昵称 | 性别 | 跟进人 | 标签 | 跟进总结 |
|---------|----------|----------|----------|------|--------|---------|----------|

| 显示列 | API 字段 | 转换规则 |
|--------|----------|----------|
| 客户 ID | `id` | 直接展示 |
| 创建时间 | `createTime` | 转为 `YYYY-MM-DD HH:mm:ss` |
| 客户名称 | `customerName` | 空值显示 `-` |
| 微信昵称 | `name` | 空值显示 `-` |
| 性别 | `gender` | `0` 显示未知，`1` 显示男，`2` 显示女 |
| 跟进人 | `userId` | 空值显示 `-` |
| 标签 | `tag` | 通过标签字典显示为“标签组/标签名”；多个标签使用逗号分隔；无法映射时显示“未匹配标签（ID）” |
| 跟进总结 | `followRecord` | 空值显示 `-`；超过 80 个字符时截断并追加 `...` |

除非用户明确要求，不展示头像 URL 和 `externalUserid`。

## 分页摘要

每次查询结束后输出：

```text
当前第 X 页 / 共 Y 页，共 Z 条客户记录
```

其中 `Y = ceil(total / pageSize)`。如需查看更多，提示用户指定页码。

## 空结果提示

```text
当前筛选条件下未找到企微客户，请调整名称、时间范围或标签条件。
```
