# 弹幕捕手企微标签列表结果模板

## 展示规则

默认按标签组展示标签列表：

| 标签组 | 标签名称 | 本地 ID | 状态 | AI 打标 |
|--------|----------|---------|------|---------|

| 显示列 | API 字段 | 转换规则 |
|--------|----------|----------|
| 标签组 | `groupName` | 空值显示 `-` |
| 标签名称 | `tagName` | 直接展示 |
| 本地 ID | `id` | 直接展示；用于客户查询的 `tagIds` 筛选 |
| 状态 | `status` | `0` 显示正常，`1` 显示已删除 |
| AI 打标 | `useAi` | `0` 显示启用，`1` 显示不启用 |

默认只展示正常标签。用户明确要求查看全部或已删除标签时，保留状态列以便区分。

## 空结果提示

```text
当前条件下未找到企微标签。
```
