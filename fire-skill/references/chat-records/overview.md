# 客户聊天记录查询

通过风火号定位 CSS 智能客服系统中的客户，再按客户 ID 与时间范围查询聊天记录。客户平台是选择正确客户的必要条件。

## 查询流程

### 工单联动

1. 从工单响应读取 `userInfoAssociatedDto.fhdCode`；缺失时请用户提供风火号。
2. 从工单的 `productIdInt` 或 `product.name` 判断 CSS 客户平台，参考下方映射。
3. 调用 [客户列表接口](customers.md)，以风火号获取客户列表。
4. 用目标平台筛选 `platform`，得到对应 `customerId`。
5. 根据用户指定的时间范围调用 [消息记录接口](messages.md)。
6. 按 [聊天记录结果模板](format-template.md) 输出时间线；不要把不同平台客户的记录混合。

### 直接查询

1. 要求用户提供风火号和时间范围。
2. 调用 [客户列表接口](customers.md)。
3. 若返回多个平台客户，询问用户要查询哪个平台；用户明确平台时再选择对应 `customerId`。
4. 调用 [消息记录接口](messages.md) 并按模板输出。

时间范围为必填项。消息接口无分页且按会话创建时间过滤；不要默认传入 `0` 或擅自扩大查询范围。
