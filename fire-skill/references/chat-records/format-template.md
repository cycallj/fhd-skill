# 客户聊天记录结果模板

## 输出原则

先给出客户摘要，再按时间正序输出消息时间线。聊天内容可能包含个人信息，只按用户指定的客户与时间范围展示，不跨平台合并记录。

```text
客户：<客户名称>（<CSS 平台>）
风火号：<FHD_CODE>
查询范围：<开始时间> 至 <结束时间>
消息数：<COUNT>
```

## 消息时间线

| 时间 | 发言方 | 消息类型 | 内容 | 状态 |
|------|--------|----------|------|------|

| 字段 | 转换规则 |
|------|----------|
| 时间 | `createTime` 转为 `YYYY-MM-DD HH:mm:ss` |
| 发言方 | `replyType=1` 显示客户；`2` 自动回复；`3` AI 回复；`4` 显示 `agentName` 或人工客服；`5` 结束语 |
| 消息类型 | 使用 `type` 枚举；图片、文件、视频仅显示类型，不输出 URL |
| 内容 | 文本消息展示 `content`；非文本或空内容显示 `-` |
| 状态 | 使用 `state` 枚举；已撤回消息标注“已撤回” |

默认不输出 `plugContent` 的 `param`、`result`，也不输出头像 URL、请求 ID 或插件原始内容。用户明确要求 AI 插件摘要时，仅展示插件名称与类型。

## 空结果提示

```text
指定客户在该时间范围内没有聊天记录。
```
